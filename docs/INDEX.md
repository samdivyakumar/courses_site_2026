## ‚ùì FAQ

### Where do I change the course fee?
Search for the course acronym (e.g., "PGDFM") in `index.html` and find the "Program Details" card.

### How do I add a new course?
Follow the template in [HTML Guide - Course Sections](HTML-GUIDE.md#course-sections).

### Why aren't my changes showing?
Clear browser cache with Ctrl+Shift+R (Cmd+Shift+R on Mac).

### How do I test mobile view?
Use Chrome DevTools Device Mode (F12, then click phone icon).

### Where are the form submissions going?
Application forms use Cognito Forms - check the form settings there.
